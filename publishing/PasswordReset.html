NAVO Registry Publishing Interface - Password Recovery


Using the NAVO registry publishing interface, you can publish new and modify existing VO resources, representing data services, catalogs, archive institutions, and many other astronomical data concepts. This system manages resources hosted at the VO registry at Space Telescope Science Institute. Resources in this registry are accessible by search tools and VO client software througout the IVOA. For more information, check the publishing help page or the main VAO web page.



If you have forgotten your password, you may request a temporary password by contacting the US VOA Community Support page. You can then change it using the form below.
